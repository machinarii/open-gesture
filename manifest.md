## Metadata Schema

| Field | Description |
|-------|-------------|
| `physical_description` | How the body physically performs the gesture (posture, hand shape, movement) |
| `intent` | The communicative purpose or message the gesture conveys |
| `emotional_state` | Overall valence: positive, neutral, or negative |
| `arousal` | Activation energy level: low, medium, or high (PAD dimensional model) |
| `emotional_adjectives` | CBT-informed list of emotions associated with the gesture (Plutchik componential model) |
| `gesture_type` | McNeill/Ekman taxonomy: emblem, illustrator-iconic, illustrator-metaphoric, illustrator-beat, illustrator-deictic, or adaptor |
| `body_parts` | Body parts involved in performing the gesture (BAP anatomical coding) |
| `number_of_people` | Participants required: single, 2 person, or 3 or more |
| `cultural_context` | Geographic/cultural scope: universal, western, east-asian, southern-european, etc. |
